Scene 8 Architecture Diagram ~15s
👆 Switch to the architecture diagram (open architecture.html in a browser tab, or show the README image)
🎙 Here's the architecture. The user speaks to the Gemini Live API for real-time voice. The ADK agent orchestrates tool calls to the Google Calendar API for reads and writes. The custom FastAPI server adds the grid analysis endpoints, and the brain rules file feeds the agent's system prompt — so policies are human-readable and editable without redeploying.
